# Azure-Jewels

Azure Jewels is a front-end jewelry e-commerce website designed with a clean and elegant aesthetic, inspired by luxury brands. The website showcases a variety of products including diamond necklaces, earrings, and bracelets, with a focus on user-friendly navigation and visual appeal.

## Features
* Modern and elegant UI with a white & blue theme
* Product showcase for jewelry collections
* Responsive design for different screen sizes
* Interactive elements using JavaScript
* Smooth navigation and structured layout

## Technologies Used
* HTML5
* CSS3
* JavaScript

## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Run the `index.html` file in your browser

## What I Learned
* Designing clean and user-friendly interfaces
* Structuring front-end projects efficiently
* Improving styling skills with CSS
* Adding interactivity using JavaScript

## Notes

This project focuses on front-end development only. No backend or database integration is included.
